# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`.

## How to add a changeset

1. Run `pnpm changeset` in the root of the repo
2. Select the packages you want to include in the changeset
3. Select the type of change (major, minor, patch)
4. Write a summary of the changes

## How to release

1. Push your changes to main (or merge a PR)
2. The GitHub Action will create a "Version Packages" PR
3. Review and merge the PR to publish to npm

For more info, see [changesets documentation](https://github.com/changesets/changesets).
